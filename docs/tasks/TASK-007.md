#### 3.1.2 查询条件

| 查询字段 | 类型 | 说明 | 默认值 |
|---------|------|------|--------|
| 班级 | 下拉选择 | 班级列表 | 全部 |
| 科目 | 下拉选择 | 科目列表（语文、数学、英语等） | 全部 |
| 考试类型 | 下拉选择 | 考试类型（期中、期末、月考、单元测试） | 全部 |
| 学号/姓名 | 文本输入 | 模糊搜索 | 空 |

#### 3.1.3 表格列定义

| 列名 | 字段 | 宽度 | 排序 | 说明 |
|------|------|------|------|------|
| 学号 | student_id | 120px | ✓ | 点击跳转学生详情 |
| 姓名 | student_name | 100px | ✓ | - |
| 班级 | class_name | 120px | ✓ | - |
| 科目 | subject | 80px | ✓ | 标签展示 |
| 考试类型 | exam_type | 100px | ✓ | 标签展示 |
| 分数 | score | 80px | ✓ | 颜色标识（及格/不及格） |
| 考试日期 | exam_date | 120px | ✓ | 日期格式化 |
| 操作 | - | 150px | - | 编辑、删除按钮 |

| 功能 | 说明 | 实现方式 |
|------|------|---------|
| 多条件查询 | 支持班级、科目、考试类型、学号/姓名组合查询 | SearchForm + API 查询参数 |
| 分页 | 支持分页展示，每页 20 条 | Pagination + API 分页参数 |
| 排序 | 支持按学号、姓名、分数等排序 | el-table sortable |
| 分数标识 | 及格（≥60）绿色，不及格（<60）红色 | 条件样式 |
| 导出 | 支持导出查询结果为 CSV | 前端导出或 API 导出 |

## 4. API 接口规范

### 4.1 成绩 API 接口

| 方法 | 路径 | 功能 | 请求参数 | 响应格式 |
|------|------|------|---------|---------|
| GET | `/api/v1/grades` | 获取成绩列表 | page, page_size, class_name, subject, exam_type, keyword | GradeListResponse |
| POST | `/api/v1/grades` | 创建成绩 | GradeCreate | Grade |
| PUT | `/api/v1/grades/{grade_id}` | 更新成绩 | GradeUpdate | Grade |
| DELETE | `/api/v1/grades/{grade_id}` | 删除成绩 | - | void |
| POST | `/api/v1/grades/batch` | 批量导入成绩 | GradeBatchCreate | BatchImportResponse |

### 4.2 请求/响应格式

#### 获取成绩列表 (GET /api/v1/grades)

**查询参数：**
- `page`: 页码（默认 1）
- `page_size`: 每页数量（默认 20，最大 100）
- `class_name`: 班级筛选（可选）
- `subject`: 科目筛选（可选）
- `exam_type`: 考试类型筛选（可选）
- `keyword`: 学号或姓名搜索（可选）

**成功响应 (200)：**
```json
{
    "success": true,
    "data": {
        "items": [
            {
                "grade_id": 1,
                "student_id": "20260001",
                "student_name": "张三",
                "class_name": "三年一班",
                "subject": "数学",
                "score": 95.0,
                "exam_type": "期中",
                "exam_date": "2026-04-15",
                "created_at": "2026-06-01T10:30:00",
                "updated_at": "2026-06-01T10:30:00"
            }
        ],
        "total": 200,
        "page": 1,
        "page_size": 20
    }
}
```

#### 创建成绩 (POST /api/v1/grades)

**请求体：**
```json
{
    "student_id": "20260001",
    "subject": "数学",
    "score": 95.0,
    "exam_type": "期中",
    "exam_date": "2026-04-15"
}
```

**成功响应 (201)：**
```json
{
    "success": true,
    "data": {
        "grade_id": 1,
        "student_id": "20260001",
        "subject": "数学",
        "score": 95.0,
        "exam_type": "期中",
        "exam_date": "2026-04-15",
        "created_at": "2026-06-01T10:30:00",
        "updated_at": "2026-06-01T10:30:00"
    }
}
```

**错误响应 (409)：**
```json
{
    "success": false,
    "error": {
        "code": "DUPLICATE",
        "message": "学生 '20260001' 的 '数学' 期中成绩已存在"
    }
}
```

#### 批量导入成绩 (POST /api/v1/grades/batch)

**请求体：**
```json
{
    "grades": [
        {
            "student_id": "20260001",
            "subject": "数学",
            "score": 95.0,
            "exam_type": "期中",
            "exam_date": "2026-04-15"
        },
        {
            "student_id": "20260002",
            "subject": "数学",
            "score": 88.5,
            "exam_type": "期中",
            "exam_date": "2026-04-15"
        }
    ]
}
```

**成功响应 (200)：**
```json
{
    "success": true,
    "data": {
        "total": 100,
        "success_count": 95,
        "fail_count": 5,
        "failed_items": [
            {
                "index": 10,
                "student_id": "20260099",
                "error": "学生不存在"
            },
            {
                "index": 25,
                "student_id": "20260001",
                "error": "成绩已存在"
            }
        ]
    }
}
```

| 重复检测 | 前后端检测逻辑需一致 | 参考后端唯一约束 |
| 学生搜索 | 大量学生数据搜索可能慢 | 使用远程搜索 + 防抖 |
| 分数验证 | 前后端验证规则需一致 | 参考 Pydantic Schema |
| 跨域问题 | API 请求可能跨域 | 配置 Vite 代理 |
