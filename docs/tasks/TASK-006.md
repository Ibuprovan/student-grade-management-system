## 4. API 接口规范

### 4.1 学生 API 接口

| 方法 | 路径 | 功能 | 请求参数 | 响应格式 |
|------|------|------|---------|---------|
| GET | `/api/v1/students` | 获取学生列表 | page, page_size, class_name, keyword | StudentListResponse |
| GET | `/api/v1/students/{student_id}` | 获取学生详情 | - | Student |
| POST | `/api/v1/students` | 创建学生 | StudentCreate | Student |
| PUT | `/api/v1/students/{student_id}` | 更新学生 | StudentUpdate | Student |
| DELETE | `/api/v1/students/{student_id}` | 删除学生 | - | void |

### 4.2 请求/响应格式

#### 获取学生列表 (GET /api/v1/students)

**查询参数：**
- `page`: 页码（默认 1）
- `page_size`: 每页数量（默认 20，最大 100）
- `class_name`: 班级筛选（可选）
- `keyword`: 搜索关键词（学号或姓名，可选）

**成功响应 (200)：**
```json
{
    "success": true,
    "data": {
        "items": [
            {
                "student_id": "20260001",
                "name": "张三",
                "gender": "男",
                "class_name": "2026级1班",
                "enrollment_year": 2026,
                "created_at": "2026-06-01T10:30:00",
                "updated_at": "2026-06-05T15:20:00"
            }
        ],
        "total": 100,
        "page": 1,
        "page_size": 20
    }
}
```

#### 创建学生 (POST /api/v1/students)

**请求体：**
```json
{
    "student_id": "20260001",
    "name": "张三",
    "gender": "男",
    "class_name": "2026级1班",
    "enrollment_year": 2026
}
```

**成功响应 (201)：**
```json
{
    "success": true,
    "data": {
        "student_id": "20260001",
        "name": "张三",
        "gender": "男",
        "class_name": "2026级1班",
        "enrollment_year": 2026,
        "created_at": "2026-06-01T10:30:00",
        "updated_at": "2026-06-01T10:30:00"
    }
}
```

**错误响应 (409)：**
```json
{
    "success": false,
    "error": {
        "code": "DUPLICATE",
        "message": "学生的学号 '20260001' 已存在"
    }
}
```
