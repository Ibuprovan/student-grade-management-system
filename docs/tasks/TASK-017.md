## 任务描述

### 需求背景
当前系统支持单个学生添加，但教务人员在学期初需要批量导入大量学生信息，手动逐个添加效率低下。需要实现"一键批量导入学生"功能，支持通过 Excel/CSV 文件批量导入学生信息。

### 功能需求
1. **文件上传界面**
   - 支持 Excel (.xlsx) 和 CSV (.csv) 文件格式
   - 提供文件拖拽上传区域
   - 文件大小限制：10MB
   - 文件格式校验

2. **导入模板下载**
   - 提供标准导入模板下载
   - 模板包含字段说明和示例数据
   - 支持 Excel 和 CSV 两种模板格式

3. **后端文件解析**
   - 解析 Excel/CSV 文件内容
   - 数据格式校验（学号格式、姓名长度、性别、班级、入学年份）
   - 批量插入数据库
   - 事务处理（全部成功或全部回滚）

4. **导入结果反馈**
   - 显示导入成功数量
   - 显示导入失败数量
   - 详细错误信息（行号、错误原因）
   - 支持下载错误报告

### 技术要求
- 后端：使用 openpyxl 解析 Excel，csv 模块解析 CSV
- 前端：使用 Element Plus 的 Upload 组件
- 文件存储：临时存储处理后删除
- 错误处理：完善的异常捕获和用户友好提示

## 验收标准

- [ ] 前端提供文件上传界面，支持 Excel/CSV 格式
- [ ] 前端提供导入模板下载功能
- [ ] 后端 API 能正确解析上传的文件
- [ ] 数据校验完整（学号格式、姓名长度、性别、班级、入学年份）
- [ ] 批量导入支持事务处理
- [ ] 导入结果反馈清晰（成功数、失败数、错误详情）

- [ ] 大文件导入性能良好（1000条记录 < 30秒）
- [ ] 错误处理完善，不因单条数据错误导致整体失败
