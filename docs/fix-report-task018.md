## 1. 问题概述

### 问题1：学生账号侧边栏显示无权限菜单

**现象：**
- 学生账号登录后，侧边栏显示了所有菜单（学生管理、成绩管理、统计分析等）
- 但学生角色只有部分权限（只能看统计概览、我的成绩）
- 点击无权限菜单后，侧边栏活跃状态会卡住

**根因：**
- `AppSidebar.vue` 中的菜单项是硬编码的，没有根据用户角色进行过滤
- 虽然路由配置中有 `roles` 限制，但侧边栏菜单显示逻辑未与角色关联

### 问题2：学生账号无法查看自己的成绩

**现象：**
- 学生账号登录后，进入"我的成绩"页面
- 由于认证问题，无法正确获取当前学生的学号
- 导致无法查询到该学生的成绩数据

**根因：**
- `User` 模型与 `Student` 模型是分离的，没有直接关联字段
- `MyGrades.vue` 使用 `authStore.user?.username` 作为 `studentId`，但用户名 ≠ 学号
- 需要通过后端接口获取当前学生用户关联的学生信息

## 5. 注意事项

### 学生用户创建规范

为确保学生账号能正常查看成绩，创建学生用户时需要遵循以下规范：

1. **用户名与学号一致**：学生的登录用户名必须与其学号相同
2. **先创建学生信息**：在创建用户账号前，先在学生管理中添加学生信息
3. **示例流程：**
   - 添加学生：学号 `20260001`，姓名 `张三`
   - 创建用户：用户名 `20260001`，密码 `xxx`，角色 `student`

### 数据关联说明

- `User.username` ↔ `Student.student_id`：通过用户名（学号）关联
- 后端接口 `/api/v1/auth/me/student-info` 通过用户名查找学生记录
- 如果用户名与学号不匹配，将返回 404 错误
