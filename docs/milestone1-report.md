## 2. Agent 角色配置

- **职责**：需求分析、用户场景设计、编写 PRD
- **输出**：`docs/prd.md`
- **配置文件**：`.opencode/agents/pm.md`

### 2.2 后端工程师 (backend-dev)
- **职责**：业务逻辑实现、核心算法与函数编写、API 实现
- **输出**：`src/` 核心代码
- **配置文件**：`.opencode/agents/backend-dev.md`

- **职责**：抽取 PRD 和代码结构，自动生成标准的实验报告
- **输出**：`reports/experiment-report.md`
- **配置文件**：`.opencode/agents/report.md`
