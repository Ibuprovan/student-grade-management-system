# V4.3.0 - 成绩录入改造与统计分析优化

> **发布日期：** 2026-06-15
> **版本类型：** 功能更新 + Bug 修复
> **主要变更：** 成绩录入/导入流程改造、统计概览重构、班级统计排序、多项 Bug 修复

---

## 更新摘要

本次更新对成绩录入、成绩导入、统计概览三个核心模块进行了全面改造，同时修复了多项显示和排序问题。

---

## 新增功能

### 1. 成绩录入全面改造
- **每人一次录入所有科目**：选中学生后，一次性填写 9 科成绩
- **实时总分计算**：填写过程中自动计算并显示总分
- **学生搜索**：支持学号/姓名模糊搜索学生
- **批量提交**：提交时自动为每个科目创建独立成绩记录
- **提交并继续**：录入完成后可继续录入下一位学生

### 2. 成绩导入模板改造
- **新模板格式**：每人一行，包含学号、姓名、9 科成绩、考试类型、考试日期
- **列定义**：`学号,姓名,语文,数学,英语,物理,化学,生物,政治,历史,地理,考试类型,考试日期`
- **留空处理**：某科未考试可留空，系统自动跳过
- **自动解析**：上传 CSV 后自动解析并预览
- **模板下载**：提供标准模板下载，含示例数据

### 3. 统计概览重构
- **总分维度统计卡片**：学生总数、平均总分、最高总分、及格率、优秀率、参考人数
- **总分分布柱状图**：按分数段展示总分分布
- **科目趋势折线图**：各科目平均分与及格率趋势
- **科目占比饼图**：各科目平均分在总分中的权重
- **能力雷达图**：平均分、及格率、优秀率三维对比
- **总分排名表**：显示全部学生的总分排名及各科明细
- **单科排名表**：可选科目，查看该科目前 10 名

### 4. 班级统计排序
- **默认排序**：按班级名称排序（一班→二班→三班→四班）
- **中文数字解析**：正确解析"一班"、"二班"等中文数字
- **可点击排序**：平均分、及格率、优秀率支持升序/降序排序
- **整行跟随**：排序时班级完整信息跟随变化

### 5. 科目统计优化
- **雷达图增强**：从二维（平均分+及格率）升级为三维（平均分+及格率+优秀率）
- **描述更新**：图表说明更清晰

## Bug 修复

| 问题 | 修复 |
|------|------|
| `el-row align="stretch"` 报错 | Element Plus 仅支持 top/middle/bottom，改用 CSS flex 实现等高 |
| 后端 `/ranking/total` 返回 500 | SQLAlchemy Numeric 类型返回 Decimal，显式转 float |
| 总分排名只显示部分学生 | 移除 limit 限制，显示全部学生 |
| 饼图中心文字与图表重叠 | graphic 定位与饼图中心对齐，内半径增大至 55% |
| 科目统计分数分布无数据 | ScoreDistribution key 与后端不匹配，改为 '0-59'...'90-100' |
| 表格未占满卡片宽度 | width 改为 min-width，添加 style="width: 100%" |
| 同行卡片高度不一致 | 使用 CSS flex 实现等高布局 |
| 按钮文字对比度不足 | 默认按钮文字颜色加深 |
| Vite 构建 manualChunks 报错 | 改为函数格式 |
