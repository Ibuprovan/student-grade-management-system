## 更新概述

全面修复 P0/P1/P2 优化项，涵盖安全增强、用户体验改进、新功能添加。

## P0 - 紧急修复

### 移除前端调试日志
- 删除 `Login.vue`、`auth.ts`、`request.ts` 中所有 `console.log('[DEBUG]...')` 语句
- 避免生产环境暴露内部逻辑

### 密码修改功能
- **后端**：新增 `POST /api/v1/auth/change-password` 端点
- **前端**：AppHeader 下拉菜单添加"修改密码"选项
- 密码强度校验：至少 8 位，包含大小写字母和数字

### JWT 密钥检查
- 生产环境（`DEBUG=false`）启动时检查 `JWT_SECRET_KEY` 是否为默认值
- 如果是默认值，拒绝启动并抛出 `RuntimeError`

### 学生查看成绩页面
- 新增 `MyGrades.vue` 页面
- 显示平均分、总分、班级/年级排名
- 成绩趋势折线图
- 各科成绩明细表格
- 仅 student 角色可见

## P1 - 重要改进

### 搜索防抖自动搜索
- `StudentList.vue`：学号/姓名输入 300ms 后自动搜索
- `GradeList.vue`：关键字输入 300ms 后自动搜索
- 班级/科目/考试类型下拉变化时立即搜索

### 导出全部筛选结果
- `StudentList.vue`：导出功能改为获取全部筛选数据后导出

### 用户管理 API
- `GET /api/v1/users` - 获取用户列表（admin）
- `POST /api/v1/users` - 创建用户（admin）
- `PUT /api/v1/users/{id}` - 更新用户（admin）
- `DELETE /api/v1/users/{id}` - 删除用户（admin）
- `PUT /api/v1/users/{id}/password` - 重置密码（admin）

### 操作审计日志
- 新增 `audit_log` 数据库表
- `GET /api/v1/audit-logs` - 查询审计日志（admin）
- 记录学生/成绩/用户的增删改操作

### 批量删除接口
- `POST /api/v1/students/batch-delete` - 批量删除学生（admin）

### 登录页隐藏默认密码
- 默认隐藏默认账号密码提示
- 添加"查看默认账号"按钮

## P2 - 改善

### 密码强度校验
- 前端统一密码强度规则
- 实时显示密码强度指示器（弱/中/强）

### Token 过期提示优化
- `POST /api/v1/auth/check-token` - 检查 Token 是否即将过期
- 返回 `expiring_soon`、`expires_at`、`remaining_minutes`
