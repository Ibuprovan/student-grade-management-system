# V2.3.2 学生账号权限与成绩查看修复

**发布日期**：2026-06-14

**版本类型**：Bug 修复

---

## 修复内容

### 1. 学生账号侧边栏权限过滤

**问题**：学生账号登录后显示所有菜单，包括无权限访问的页面

**修复**：根据用户角色动态显示侧边栏菜单项

**权限矩阵**：

| 菜单项 | admin | teacher | student |
|--------|:-----:|:-------:|:-------:|
| 仪表盘 | ✅ | ✅ | ✅ |
| 学生管理 | ✅ | ✅ | ❌ |
| 成绩管理 | ✅ | ✅ | ❌ |
| 统计概览 | ✅ | ✅ | ✅ |
| 班级统计 | ✅ | ✅ | ❌ |
| 科目统计 | ✅ | ✅ | ❌ |
| 我的成绩 | ❌ | ❌ | ✅ |

### 2. 学生账号成绩查看修复

**问题**：学生账号无法查看自己的成绩，认证信息中缺少学号

**修复**：
- 后端新增 `GET /api/v1/auth/me/student-info` 接口
- 获取当前学生用户关联的学生信息（学号、姓名、班级等）
- 前端"我的成绩"页面使用新接口自动获取学生信息

**学生用户创建规范**：
1. 学生的登录用户名必须与其学号相同
2. 先在学生管理中添加学生信息，再创建用户账号
3. 示例：学号 `20260001` → 用户名 `20260001`
