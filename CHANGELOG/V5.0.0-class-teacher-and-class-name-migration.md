# V5.0.0 - 班主任角色上线与班级名称统一

> **发布日期：** 2026-06-19
> **版本类型：** 重大功能更新
> **主要变更：** 新增班主任（class_teacher）角色、统一班级名称为"2026级X班"格式、多项前端修复

---

## 更新摘要

本次更新新增了**班主任角色**，支持管理员分配班主任并自动创建账号，班主任可查看本班学生、成绩与统计数据（只读）。同时将全局班级名称从"三年X班"统一为"2026级X班"格式，确保与学号编码规则一致。

---

## 新增功能

### 1. 班主任（Class Teacher）角色
- **完整 RBAC 角色**：`class_teacher` 角色，与 admin/teacher/student 并列
- **自动创建账号**：管理员分配班主任时，系统自动创建用户账号
- **账号规则**：用户名 = `{入学年份}{班级编号 3 位}`（如 2026001），初始密码 `123456`，首次登录强制修改密码
- **四大专属页面**：
  - **仪表盘**：班级学生数、成绩记录数、平均总分、及格率、优秀率
  - **学生列表**：仅展示本班学生，支持搜索/筛选/分页
  - **成绩列表**：仅展示本班学生成绩，支持科目/日期筛选、总分排名
  - **统计分析**：统计概览（总分统计 + 排名） + 科目统计（图表 + 分数分布）
- **只读权限**：班主任只能查看数据，不能增删改

### 2. 管理员 - 班主任管理
- **新增班主任**：从已有班级列表中选择，自动检测是否已分配
- **删除班主任**：同时删除关联的用户账号
- **班主任列表**：查看所有班主任及其班级信息

### 3. 班级名称格式统一
- **新格式**：`{入学年份}级{班级编号}班`（如 `2026级1班`）
- **数据库迁移**：已执行迁移脚本，所有学生记录从"三年X班"→"2026级X班"
- **全端同步**：后端代码、前端页面、测试数据、文档全部更新
- **排序优化**：按班级编号数字排序，无需中文数字解析

### 4. 成绩显示优化（MyGrades）
- **总分保留 1 位小数**：与录入精度一致
- **通过率/优秀率卡片**：替换原"平均分"卡片，显示 ≥60 及格人数、≥90 优秀人数
- **表格全宽**：使用 `min-width` 确保表格占满容器宽度
- **标准说明**：添加 "≥60 为及格，≥90 为优秀" 提示

### 5. 科目分数分布图表修复
- **动态分数段**：主科（语数外，满分 150）使用 `0-89,90-104,105-119,120-134,135-150`
- **其他科目**：使用 `0-59,60-69,70-79,80-89,90-100`
- **筛选联动**：选择不同科目时图表正确展示对应分数段
