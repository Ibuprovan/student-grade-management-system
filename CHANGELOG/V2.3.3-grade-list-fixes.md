# V2.3.3 成绩列表全面修复

**发布日期**：2026-06-14

**版本类型**：Bug 修复

---

## 修复内容

### 1. 修复成绩列表不显示学生姓名和班级

**问题**：成绩列表只显示学号，不显示学生姓名和班级

**原因**：后端 Grade 模型有 `student` 关联对象，但 `GradeResponse` schema 期望 `student_name` 和 `class_name` 作为顶层字段

**修复**：
- 添加 `_build_grade_response()` 和 `_build_grade_responses()` 辅助函数
- 从 `grade.student` 提取 `name` 和 `class_name` 填充到响应中
- 所有成绩接口统一使用辅助函数

### 2. 修复成绩列表 ElTag type 警告

**问题**：控制台报错 `Invalid prop: validation failed for prop "type"`

**原因**：`getExamTypeTag` 函数对"期中"返回空字符串 `''`，ElTag 不接受

**修复**：空字符串改为 `'primary'`，确保返回值都是有效类型

### 3. 修复成绩导出失败 (422 错误)

**问题**：点击导出按钮后报 422 错误

**原因**：导出时 `page_size=10000` 超过后端限制 (最大100)

**修复**：改为分页获取所有数据，每次获取100条，循环直到获取完毕

### 4. 修复成绩导出数据字段错误

**问题**：导出的 CSV 中姓名和班级显示为空

**原因**：使用嵌套的 `student` 对象，但后端返回的是 `student_name` 和 `class_name`

**修复**：改为使用 `student_name` 和 `class_name` 字段

### 5. 修复侧边栏活跃状态不更新

**问题**：点击侧边栏菜单时，绿色活跃指示器不移动

**原因**：`el-menu` 使用 `default-active` 只设置初始状态

**修复**：
- 添加 `model-value` 实现响应式更新
- 添加 import 页面的活跃状态映射
