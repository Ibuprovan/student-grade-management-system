# V4.1.0 - 布局修复与数据可视化优化

> **发布日期：** 2026-06-15
> **版本类型：** 功能修复
> **主要变更：** 全局布局修复、图表容器规范化、统计数据修正

---

## 更新摘要

本次更新修复了 V4.0.0 专业教育风格 UI 重设计后遗留的多项布局与数据展示问题，涵盖图表溢出、卡片对齐、表格宽度、侧边栏折叠、响应式断点、饼图重叠、科目统计分数分布等多个层面。

---

## Bug 修复

### 1. 科目统计分数分布图无数据
- **问题**：科目统计页分数分布柱状图始终显示为空，无论是否选中科目
- **根因**：后端 `score_distribution` 返回 key 为 `"0-59"`、`"60-69"` 等字符串，前端 TypeScript 接口定义 key 为 `fail`、`pass` 等，取值全部为 `undefined`
- **修复**：
  - `ScoreDistribution` 接口 key 改为与后端一致的 `'0-59'`...`'90-100'`
  - `scoreDistributionData` 计算属性改用 `dist['0-59']` 取值
  - 未选中科目时自动聚合全部科目的分数分布作为默认展示

### 2. 统计概览饼图中心文字与图表重叠
- **问题**：科目平均分占比饼图的中心标题"科目分布"与环形图区域重叠
- **根因**：graphic 定位在 `left: 'center'`（50%），但饼图 `center` 设为 `['40%', '50%']`，水平不对齐；内半径 50% 过小
- **修复**：graphic `left` 改为 `'40%'` 与饼图中心对齐，字号 18px→14px，内半径 50%→55%

### 3. 图表覆盖文字 / 内容溢出
- **修复**：所有图表组件（BarChart、LineChart、PieChart、RadarChart）添加 `overflow: hidden`
- **修复**：图表 grid 间距增大（left 50→60, right 30→40, bottom 50→60），防止坐标轴标签被裁切

### 4. 表格未占满卡片宽度
- **问题**：成绩列表、学生列表等表格固定 `width` 导致右侧大片空白
- **修复**：所有 `el-table-column` 从 `width` 改为 `min-width`，表格添加 `style="width: 100%"`

### 5. 同行卡片高度不一致
- **问题**：统计页面同一 `el-row` 内的图表卡片高度参差不齐
- **修复**：`el-row[align="stretch"]` 内的 `el-col` 设 `display: flex`，`.page-card` 设 `flex: 1`

### 6. 侧边栏折叠后图标布局问题
- **修复**：折叠状态隐藏底部版权文字、隐藏子菜单箭头，图标 20px 居中显示

### 7. 主内容区宽度限制
- **问题**：`.page-container` 和 `.layout-content` 硬编码 `max-width: 1440px`，大屏下内容缩在中间
- **修复**：移除 `max-width` 限制，改为 `width: 100%`，内容自适应屏幕宽度

### 8. Vite 构建配置错误
- **问题**：`manualChunks` 使用对象格式，Vite 8.x 要求函数格式
- **修复**：改为函数形式按模块路径分包

## 改进项

### 图表容器统一
- 所有图表默认高度从 300px/350px 统一为 **360px**
- 图表容器统一添加 `overflow: hidden` 防止内容溢出

### 排序指示器
- 表格排序三角标添加明确的颜色样式（默认灰色、激活时专业蓝）
- 可排序列添加 hover 高亮效果

### 按钮对比度
- 默认按钮文字颜色从 `$text-secondary`（#6b7280）加深为 `$text-primary`（#1f2937）

### 排版规范
- 全局 `line-height` 统一为 1.55
- 标题层级：h1=24px, h2=20px, h3=16px, h4=14px

### z-index 层级管理
- `.el-popper` z-index 2000
- `.el-dialog__wrapper` z-index 2100
- `.el-message` / `.el-notification` z-index 3000

### 响应式断点
- 新增 1280px、1440px、1920px 三个断点的 padding 适配
