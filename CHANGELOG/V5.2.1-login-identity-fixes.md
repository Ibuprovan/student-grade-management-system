# V5.2.1 - 身份标签修复与默认账号清理

> **发布日期：** 2026-06-19
> **版本类型：** Bug 修复 + 清理
> **主要变更：** 修复班主任/学科组长右上角身份显示、调整侧边栏顺序、删除默认 teacher/student 账号、移除登录界面默认账号提示

---

## 更新摘要

本次更新修复了班主任和学科组长登录后右上角显示「未知角色」的问题，调整了管理员端侧边栏的板块顺序，同时清除了系统中残留的 teacher/student 默认账号相关代码与提示。

---

## Bug 修复

| 问题 | 修复 |
|------|------|
| 班主任登录后右上角显示「未知角色」 | `roleLabel` 添加 `class_teacher: '班主任'` |
| 学科组长登录后右上角显示「未知角色」 | `roleLabel` 添加 `subject_leader: '学科组长'` |
| 班主任/学科组长头像颜色与标签样式未区分 | `avatarColor` 和 `roleTagType` 分别添加对应角色配置 |

## 清理

### 删除默认 teacher/student 账号
- **`init_users.py`**：`DEFAULT_USERS` 仅保留 admin，移除 teacher 和 student
- **`start.bat`** / **`start.ps1`**：默认账户显示仅保留 `admin / admin123`
- **`README.md`** / **`USER_GUIDE.md`**：默认账户表仅保留管理员行，添加说明文字

### 移除登录界面默认账号提示
- **`Login.vue`**：删除「默认账号：admin / admin123」提示框、「查看默认账号」按钮及相关 CSS

## 侧边栏调整

管理员端「账号管理」板块从原位置（班主任管理之后）移至「统计分析」板块下方，新顺序为：
```
仪表盘 → 学生管理 → 成绩管理 → 统计分析 → 账号管理 → 班主任管理 → 学科组长管理
```
