# V2.1.2 - 登录与统计页面修复

## 更新概述

修复登录功能和统计页面的多个阻断性 Bug，确保系统核心功能正常可用。

## 登录功能修复

| 编号 | 问题 | 根因 | 修复 |
|------|------|------|------|
| FIX-001 | 登录按钮点击无响应 | `formRef.value` 为空，`FormInstance` 是 type-only import | 显式导入 `ElForm` 组件，使用 `InstanceType<typeof ElForm>` |
| FIX-002 | Element Plus 图标无法解析 | 图标组件未全局注册 | 在 `main.ts` 中全局注册所有 Element Plus 图标 |
| FIX-003 | `UserPlus` 图标不存在 | Element Plus 图标库中无此图标 | 改为 `CirclePlus` |
| FIX-004 | `PieChart` 图标冲突 | 自定义图表组件与图标名冲突 | 改为 `TrendCharts` |

## API 路由修复

| 编号 | 问题 | 根因 | 修复 |
|------|------|------|------|
| FIX-005 | 班级列表 422 错误 | `page_size: 1000` 超出后端限制 100 | 改用专用 `/students/classes` 接口 |
| FIX-006 | 成绩列表 405 错误 | 前端调用 `/grades`，后端路由是 `/grades/search` | 前端改为 `/grades/search` |
| FIX-007 | 统计排名 404 错误 | 前端调用 `/statistics/ranking`，后端是 `/ranking/subject` | 前端改为正确路径 |

## 数据解包修复

| 编号 | 问题 | 根因 | 修复 |
|------|------|------|------|
| FIX-008 | 学生列表 `items` undefined | 响应解包错误，直接访问 `response.items` | 改为 `response.data.items` |
| FIX-009 | 统计数据 `average` undefined | 响应解包错误 | 添加 `?.` 可选链保护 |
| FIX-010 | 科目统计 `subjects.map` undefined | 批量数据解包错误 | 正确解包 + 空数据兜底 |

## 空数据保护

| 编号 | 问题 | 修复 |
|------|------|------|
| FIX-011 | RadarChart ECharts 空数据崩溃 | 空数据时跳过初始化，显示"暂无数据" |
| FIX-012 | DataTable data undefined | 添加默认值 `() => []` |
| FIX-013 | 统计页面空白 | 所有图表和表格添加空数据保护 |
