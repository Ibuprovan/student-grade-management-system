> **日期：** 2026-06-07

## 更新概述

完成系统第二代全面升级，新增用户认证系统、UI美化、性能优化和安全增强。

## 新增功能

### 用户认证系统
- 完整的登录页面（`frontend/src/views/login/Login.vue`）
- JWT 双令牌机制（Access Token + Refresh Token）
- 路由守卫与权限控制（`frontend/src/router/index.ts`）
- Token 自动刷新机制（`frontend/src/utils/request.ts`）
- 认证状态管理（`frontend/src/stores/auth.ts`）
- 认证 API 封装（`frontend/src/api/auth.ts`）

### Dashboard 真实数据
- 后端 Dashboard API（`src/api/routes/dashboard.py`）
- 后端 Dashboard Service（`src/services/dashboard_service.py`）
- 前端 Dashboard Store（`frontend/src/stores/dashboard.ts`）
- Dashboard 页面接入真实数据

### UI 美化 - 温馨友好风格
- 全局样式重构（`frontend/src/assets/styles/global.scss`）
- 侧边栏美化（深蓝青背景、Logo渐变图标）
- 登录页面美化（渐变深色背景、卡片入场动画）
- Dashboard 美化（统计卡片圆角图标）
- 列表页面美化（表格、搜索区域优化）
- 表单页面美化（圆角、输入框样式）
- 错误页面美化（403、404）

### 权限控制
- v-permission 指令（`frontend/src/directives/permission.ts`）
- 角色权限控制（admin/teacher/student）
- 路由级别权限控制

### 安全增强
- 速率限制：登录接口 10次/分钟（slowapi）
- 批量限制：成绩批量创建最多 500 条
- 安全响应头配置

## 优化改进

### 后端性能优化
- 搜索学生：内存分页改为数据库分页
- 班级列表：新增独立接口，使用 DISTINCT 查询
- Dashboard Service：新增统计汇总服务

### 用户体验优化
- AppHeader 显示真实用户信息
- 退出登录功能完整实现
- 用户角色标签显示（admin/teacher/student）
- 头像显示用户名首字母，根据角色显示不同颜色
