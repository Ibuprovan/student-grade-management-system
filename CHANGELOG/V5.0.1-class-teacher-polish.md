# V5.0.1 - 班主任功能优化与修复

> **发布日期：** 2026-06-19
> **版本类型：** Bug 修复 + 功能优化
> **主要变更：** 班主任对话框修复、自动生成成绩汇总、科目排序统一、启动脚本修复

---

## 更新摘要

本次更新针对 V5.0.0 新增的班主任角色进行多项修复与优化：修复对话框数据加载时序问题，自动生成班级成绩汇总数据，删除自动初始化脚本改为手动管理，统一全局科目排序，优化班主任科目统计页面。

---

## Bug 修复

| 问题 | 修复 |
|------|------|
| 班主任管理对话框打开时下拉框为空 | 重写对话框：使用 `v-if` + `destroy-on-close`，确保数据加载后才渲染 |
| 添加班主任对话框数据加载时序错误 | 移除 `watch`/`nextTick`，改为顺序 fetch-then-open |
| 新增班主任后班级下拉无数据 | 使用 `ensureAdminSession()` 在请求前主动校验管理员会话 |
| 新增班主任 + 预选第一个班级 | 打开对话框时自动选中第一个可分配班级 |
| `start.bat` 中文路径编码错误 | 通道编码改为 `chcp 65001`，使用 `/D` 标志设置工作目录 |

## 功能优化

### 1. 成绩汇总自动生成
- **新增 `_ensure_exam_totals()`**：创建班主任时自动为该班级所有学生生成 `student_exam_totals` 数据
- **删除自动初始化脚本**：`init_class_teachers.py` 删除，改为管理员端手动添加
- **新增 `regenerate_exam_totals.py`**：实用脚本，可重新生成所有学生的成绩汇总

### 2. 科目排序统一
- **全局排序**：所有科目统一按 `语 → 数 → 英 → 物 → 化 → 生 → 政 → 史 → 地` 排序
- **`subject_order_map`**：新增排序映射字典，替代 `sorted()` 函数
- **跨端同步**：后端 API、前端页面、CLI 工具、文档全部更新

### 3. 班主任成绩列表优化
- **总分视图**：成绩列表改为总分排名视图，展示学生总分、各科成绩、排名
- **科目筛选**：支持按科目过滤成绩列表
- **统计优化**：班主任科目统计添加科目筛选功能

### 4. 启动脚本增强
- **`start.bat` 重构**：修复中文路径编码问题，使用 `chcp 65001` UTF-8 编码
- **工作目录**：使用 `/D` 标志设置工作目录，避免中文路径嵌套引号问题
- **`start.ps1` 同步**：PowerShell 启动脚本同步更新
