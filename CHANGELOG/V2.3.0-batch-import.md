## 更新内容

### 1. 批量导入学生功能

**新增功能**：支持通过 Excel/CSV 文件一键导入多位学生

#### 后端实现

**新增文件**：
- `src/services/import_service.py` - 批量导入服务
- `src/api/routes/imports.py` - 导入 API 路由
- `src/schemas/import_schema.py` - Pydantic 数据模式

**功能特性**：
- 支持 Excel (.xlsx) 和 CSV 格式
- 文件大小限制：10MB
- 数据预览：导入前预览数据，显示有效/无效统计
- 数据校验：完整字段校验（姓名、学号、班级、性别等）
- 重复检测：学号唯一性检查
- 事务处理：批量插入使用事务，保证数据一致性
- 错误反馈：详细错误信息，包含行号和错误原因

**API 端点**：
| 方法 | 端点 | 说明 |
|------|------|------|
| POST | `/api/v1/import/preview` | 预览导入数据 |
| POST | `/api/v1/import/students` | 执行批量导入 |
| GET | `/api/v1/import/template/{format}` | 下载导入模板 |

#### 前端实现

**新增文件**：
- `frontend/src/views/student/StudentImport.vue` - 批量导入页面
- `frontend/src/api/import.ts` - 导入 API 封装

**功能特性**：
- 文件上传：拖拽或点击上传 Excel/CSV 文件
- 数据预览：表格展示预览数据，高亮无效行
- 统计信息：显示总行数、有效行数、无效行数
- 模板下载：支持下载 Excel/CSV 模板
- 导入结果：成功/失败统计，可展开查看错误详情
- 进度反馈：导入过程中显示加载状态

### 2. 路由更新

- 在学生管理菜单中添加"批量导入"入口

## 使用说明

1. 访问学生管理页面
2. 点击"批量导入"按钮
3. 下载导入模板（可选）
4. 填写学生信息
5. 上传文件
6. 预览数据，确认无误
7. 点击"确认导入"
8. 查看导入结果
