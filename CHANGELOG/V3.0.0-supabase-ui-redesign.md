## 🔧 技术实现

### 设计令牌系统
```scss
// 颜色变量
$bg-primary: #0f0f23;
$bg-secondary: #1a1a2e;
$accent-primary: #3ecf8e;

// 字体变量
$font-sans: 'Geist Sans', sans-serif;
$font-mono: 'Geist Mono', monospace;

// 间距变量
$space-4: 1rem;
$space-6: 1.5rem;
```

### Element Plus覆盖
```scss
:root {
  --el-color-primary: #{$accent-primary};
  --el-bg-color: #{$bg-secondary};
  --el-text-color-primary: #{$text-primary};
  // ... 更多变量覆盖
}
```
