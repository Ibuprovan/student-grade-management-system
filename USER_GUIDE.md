# 学生成绩管理系统 - 用户手册指南

## 📖 项目简介

这是一个**学生成绩管理系统**，可以帮助老师和学校管理人员轻松管理学生的基本信息和考试成绩。系统提供了以下功能：

- **学生管理**：添加、修改、删除和查询学生信息
- **成绩管理**：录入、修改、删除和查询学生的考试成绩
- **统计分析**：查看班级成绩统计、科目成绩统计、学生个人成绩分析等
- **数据可视化**：通过图表直观展示成绩分布和趋势

## 🖥️ 系统要求

在安装和使用本系统前，请确保您的电脑满足以下要求：

### 必需软件
1. **Python 3.9 或更高版本** - 用于运行后端服务
2. **Node.js 16 或更高版本** - 用于运行前端界面
3. **Git** - 用于下载项目代码（可选，也可以直接下载压缩包）

### 操作系统
- Windows 10/11
- macOS 10.15 或更高版本
- Linux（Ubuntu 20.04 或更高版本）

## 🚀 安装步骤

**提示**：如果您使用 Windows 系统，可以跳过以下手动安装步骤，直接使用 `start.bat` 一键启动脚本，它会自动完成依赖安装和数据库初始化。

### 第一步：下载项目代码

#### 方法一：使用 Git 下载（推荐）
1. 打开命令提示符（Windows）或终端（macOS/Linux）
2. 输入以下命令下载项目：
   ```bash
   git clone https://github.com/Ibuprovan/student-grade-management-system.git
   ```
3. 进入项目目录：
   ```bash
   cd student-grade-management-system
   ```

#### 方法二：直接下载压缩包
1. 访问项目页面：https://github.com/Ibuprovan/student-grade-management-system
2. 点击绿色的 "Code" 按钮
3. 选择 "Download ZIP"
4. 解压下载的压缩包到任意位置

### 第二步：安装后端依赖

1. 打开命令提示符或终端
2. 进入项目目录
3. 安装 Python 依赖包：
   ```bash
   pip install -r requirements.txt
   ```

### 第三步：安装前端依赖

1. 进入前端目录：
   ```bash
   cd frontend
   ```
2. 安装 Node.js 依赖包：
   ```bash
   npm install
   ```
3. 返回项目根目录：
   ```bash
   cd ..
   ```

### 第四步：初始化数据库

1. 在项目根目录运行以下命令：
   ```bash
   python -m src.scripts.init_users
   ```
2. 这将创建数据库并添加默认用户账户

## 🏃‍♂️ 启动系统

### 一键启动（推荐）

**Windows 用户**：双击运行项目根目录下的 `start.bat` 文件，系统将自动：
1. 检查并安装所需依赖（Python 和 Node.js）
2. 初始化数据库并创建默认用户
3. 启动后端和前端服务
4. 自动打开浏览器访问系统

**其他系统用户**：请参考下方手动启动步骤。

### 手动启动

#### 启动后端服务

1. 在项目根目录运行：
   ```bash
   uvicorn src.main:app --reload --host 0.0.0.0 --port 8000
   ```
2. 看到类似以下信息表示启动成功：
   ```
   INFO:     Uvicorn running on http://0.0.0.0:8000 (Press CTRL+C to quit)
   INFO:     Started reloader process [xxxxx]
   ```
3. **保持此窗口打开**，不要关闭

### 启动前端界面

1. 打开**新的**命令提示符或终端窗口
2. 进入前端目录：
   ```bash
   cd frontend
   ```
3. 启动开发服务器：
   ```bash
   npm run dev
   ```
4. 看到类似以下信息表示启动成功：
   ```
   VITE v4.x.x  ready in xxx ms
   ➜  Local:   http://localhost:5173/
   ➜  Network: http://192.168.x.x:5173/
   ```

### 访问系统

1. 打开浏览器（推荐使用 Chrome、Firefox 或 Edge）
2. 访问地址：**http://localhost:5173**
3. 您将看到系统的登录页面

## 📚 基本操作指南

### 1. 登录系统
1. 在登录页面输入用户名和密码
2. 点击"登录"按钮
3. 登录成功后，您将看到系统主界面

### 2. 学生管理
#### 添加学生
1. 点击左侧菜单的"学生管理"
2. 点击"添加学生"按钮
3. 填写学生信息：
   - **学号**：8位数字，格式为年份+4位序号（如：20230001）
   - **姓名**：学生真实姓名
   - **性别**：选择"男"或"女"
   - **班级**：如"计算机科学1班"
   - **入学年份**：如2023
4. 点击"保存"按钮

#### 查询学生
1. 在学生管理页面，使用搜索框输入学号或姓名
2. 点击"搜索"按钮
3. 系统将显示匹配的学生列表

#### 修改学生信息
1. 在学生列表中找到要修改的学生
2. 点击该学生所在行的"编辑"按钮
3. 修改需要更新的信息
4. 点击"保存"按钮

#### 删除学生
1. 在学生列表中找到要删除的学生
2. 点击该学生所在行的"删除"按钮
3. 在确认对话框中点击"确定"
4. **注意**：删除学生会同时删除该学生的所有成绩记录

### 3. 成绩管理
#### 录入成绩
1. 点击左侧菜单的"成绩管理"
2. 点击"录入成绩"按钮
3. 填写成绩信息：
   - **学号**：输入学生学号（系统会自动显示学生姓名）
   - **科目**：如"数学"、"语文"、"英语"等
   - **分数**：0-100之间的数字，支持小数（如85.5）
   - **考试类型**：如"期中考试"、"期末考试"、"单元测试"
   - **考试日期**：选择考试日期
4. 点击"保存"按钮

#### 批量录入成绩
1. 在成绩管理页面，点击"批量录入"按钮
2. 下载成绩模板文件（Excel格式）
3. 按照模板格式填写成绩数据
4. 上传填写好的文件
5. 系统将自动导入成绩

#### 查询成绩
1. 在成绩管理页面，使用筛选条件：
   - 按学生查询：输入学号
   - 按班级查询：选择班级
   - 按科目查询：选择科目
   - 按考试类型查询：选择考试类型
2. 点击"查询"按钮
3. 系统将显示符合条件的成绩列表

### 4. 统计分析
#### 查看班级统计
1. 点击左侧菜单的"统计分析"
2. 选择"班级统计"选项卡
3. 选择要查看的班级和科目
4. 系统将显示以下统计信息：
   - 平均分、最高分、最低分
   - 及格率、优秀率
   - 分数段分布（图表展示）

#### 查看科目统计
1. 在统计分析页面，选择"科目统计"选项卡
2. 选择要查看的科目
3. 系统将显示该科目在所有班级的统计对比

#### 查看学生个人统计
1. 在统计分析页面，选择"学生统计"选项卡
2. 输入学生学号
3. 系统将显示该学生的所有成绩统计：
   - 各科成绩对比
   - 成绩趋势分析
   - 班级排名情况

### 5. 数据导出
1. 在任意列表页面，点击"导出"按钮
2. 选择导出格式（CSV或Excel）
3. 选择要导出的数据范围
4. 点击"确定"按钮
5. 文件将自动下载到您的电脑

## ❓ 常见问题解答

### Q1: 忘记密码怎么办？
A: 目前系统不支持自助找回密码。请联系系统管理员重置密码。

### Q2: 为什么无法登录？
A: 请检查：
1. 用户名和密码是否正确（注意大小写）
2. 后端服务是否正常运行（检查第一个命令窗口）
3. 前端服务是否正常运行（检查第二个命令窗口）

### Q3: 如何修改默认端口？
A: 
- 修改后端端口：启动命令中修改 `--port` 参数，如 `--port 8001`
- 修改前端端口：编辑 `frontend/vite.config.ts` 文件中的 `server.port` 配置

### Q4: 数据会丢失吗？
A: 系统使用 SQLite 数据库，数据保存在 `data/grades.db` 文件中。请定期备份此文件以防数据丢失。

### Q5: 如何备份数据？
A: 复制 `data/grades.db` 文件到安全位置即可完成备份。

### Q6: 支持哪些浏览器？
A: 推荐使用以下浏览器的最新版本：
- Google Chrome
- Mozilla Firefox
- Microsoft Edge
- Apple Safari

### Q7: 如何报告问题或建议？
A: 请访问项目的 GitHub 页面：https://github.com/Ibuprovan/student-grade-management-system/issues
点击 "New issue" 按钮提交问题或建议。

## 🔄 更新系统

当有新版本发布时，请按以下步骤更新：

1. 备份数据：复制 `data/grades.db` 文件到安全位置
2. 下载新代码：使用 `git pull` 命令或重新下载压缩包
3. 更新依赖：
   ```bash
   pip install -r requirements.txt
   cd frontend
   npm install
   cd ..
   ```
4. 重启服务：关闭所有运行窗口，重新按照"启动系统"步骤操作

## 📝 使用技巧

1. **定期备份**：每周至少备份一次 `data/grades.db` 文件
2. **使用强密码**：修改默认密码，使用包含字母、数字和特殊字符的强密码
3. **及时保存**：录入数据后及时点击保存按钮
4. **使用筛选**：在大量数据中使用筛选功能快速找到目标信息
5. **查看图表**：统计图表比数字更直观，多使用图表分析功能
