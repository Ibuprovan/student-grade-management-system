# 学生成绩管理系统 - 用户手册

## 项目简介

学生成绩管理系统是一个现代化的全栈 Web 应用，帮助教师和学校管理人员高效管理学生信息和考试成绩。系统采用专业教育风格浅色主题，界面清晰现代。

### 主要功能
- **用户认证**：JWT 双令牌认证，支持多角色权限控制
- **学生管理**：添加、修改、删除和查询学生信息
- **成绩管理**：录入、修改、删除和查询学生的考试成绩
- **统计分析**：班级统计、科目统计、数据可视化图表
- **仪表盘**：展示系统关键指标和统计数据
- **批量导入**：支持 Excel/CSV 文件批量导入学生和成绩

## 安装步骤

### 第一步：下载项目

```bash
git clone https://github.com/Ibuprovan/student-grade-management-system.git
cd student-grade-management-system
```

### 第二步：安装依赖

```bash
# 后端依赖
pip install -r requirements.txt

# 前端依赖
cd frontend
npm install
cd ..
```

### 第三步：初始化数据库

```bash
python -m src.scripts.init_users
```

> Windows 用户可直接运行 `start.bat` 一键完成以上所有步骤。

---

## 启动系统

### 一键启动（Windows）

双击 `start.bat`，系统将自动完成依赖安装、数据库初始化、服务启动并打开浏览器。

### 手动启动

```bash
# 终端 1：启动后端
uvicorn src.main:app --reload --host 0.0.0.0 --port 8000

# 终端 2：启动前端
cd frontend
npm run dev
```

访问 **http://localhost:5173** 进入系统。

## 操作指南

### 1. 登录系统

1. 打开浏览器访问 http://localhost:5173
2. 输入用户名和密码
3. 点击"登录"按钮
4. 系统会自动记住登录状态，Token 过期时自动刷新

### 2. 学生管理

**添加学生**：
1. 左侧菜单点击"学生管理" → "添加学生"
2. 填写学号（8位数字，如 20230001）、姓名、性别、班级、入学年份
3. 点击"保存"

**查询学生**：
1. 在学生列表页使用搜索框输入学号或姓名
2. 支持按班级下拉筛选
3. 点击列标题可排序

**批量导入**：
1. 点击"批量导入"
2. 下载模板文件，按格式填写
3. 上传文件，系统自动导入

### 3. 成绩管理

**录入成绩**：
1. 左侧菜单点击"成绩管理" → "成绩录入"
2. 选择考试类型和考试日期
3. 搜索并选择学生（支持学号/姓名搜索）
4. 填写各科成绩（0-100），系统自动计算总分
5. 点击"提交"或"提交并继续"

**查询成绩**：
1. 在成绩列表页使用多维度筛选：班级、科目、考试类型、学号/姓名
2. 支持按学号、分数、日期排序
3. 切换到"总分排名"标签页查看总分排名

**批量导入**：
1. 点击"批量导入"
2. 下载模板（每人一行：学号+姓名+9科成绩+考试类型+日期）
3. 填写后上传，系统自动解析并导入

### 4. 统计分析

**统计概览**：
- 总分统计卡片（学生总数、平均总分、最高总分、及格率、优秀率）
- 总分分布柱状图、各科目趋势折线图
- 科目平均分占比饼图、三维能力雷达图（平均分+及格率+优秀率）
- 总分排名表（全部学生）、单科排名表（可选科目）

**班级统计**：
- 选择班级和考试类型后查询
- 查看班级概览卡片（学生人数、平均分、及格率）
- 班级统计表格（按班级名称排序，支持平均分/及格率/优秀率排序）
- 班级平均分对比柱状图、及格率/优秀率折线图

**科目统计**：
- 选择科目和考试类型后查询
- 查看科目概览卡片
- 科目统计表格（可点击行查看该科目分数分布）
- 科目对比柱状图、及格率/优秀率折线图
- 分数分布柱状图、能力雷达图

### 5. 数据导出

在任意列表页面点击"导出"按钮，系统将生成 CSV 文件并自动下载。

### 6. 修改密码

1. 点击右上角用户头像
2. 选择"修改密码"
3. 输入旧密码和新密码（至少8位，含大小写字母和数字）
4. 点击"确认修改"

## 常见问题

**Q: 忘记密码怎么办？**
A: 请联系系统管理员重置密码。

**Q: 为什么无法登录？**
A: 请检查用户名密码是否正确、后端服务是否正常运行。

**Q: 数据会丢失吗？**
A: 数据保存在 `data/grades.db` 文件中，请定期备份此文件。

**Q: 如何备份数据？**
A: 复制 `data/grades.db` 文件到安全位置即可。

**Q: 如何修改端口？**
A: 后端修改启动命令的 `--port` 参数；前端编辑 `frontend/vite.config.ts` 中的 `server.port`。
